# Web Ramadhan 1447H

Selamat datang di proyek **Web Ramadhan**! 🌙

Proyek ini merupakan hasil dari tantangan intensif membuat desain web selama **15 hari** yang dilaksanakan tepat di bulan suci Ramadhan 1447H. Proyek ini diselenggarakan oleh **Alhazen Academy**.

## Tentang Proyek
Tujuan dari proyek ini adalah untuk mengasah kemampuan pengembangan web (frontend) dengan membuat berbagai fitur yang bermanfaat selama bulan Ramadhan. Dalam proses pengembangannya, saya memanfaatkan bantuan **AI Gemini** sebagai asisten untuk menulis kode, memecahkan masalah (*debugging*), dan memberikan ide desain.

## Fitur-Fitur
Website ini mencakup beberapa halaman fungsional, di antaranya:
* **Portal Ramadhan**: Halaman utama untuk navigasi fitur.
* **Doa Sahur & Buka**: Kumpulan doa-doa harian.
* **Dzikir Harian**: Panduan dzikir yang bisa dibaca setiap hari.
* **Jadwal Imsakiyah**: Informasi waktu imsak dan berbuka.
* **Kalkulator Zakat**: Membantu perhitungan zakat fitrah/mal secara cepat.
* **To-Do List Ramadhan**: Daftar kegiatan untuk produktivitas ibadah.

## Teknologi yang Digunakan
* **HTML5**
* **CSS3**
* **JavaScript**
* **AI Tools**: [Gemini AI](https://gemini.google.com/)
